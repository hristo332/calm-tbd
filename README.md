# C.A.L.M. × TBD™ 90-Day Dashboard

Widget за проследяване на прогреса в 90-дневния TBD™ Road Map.

## Инсталация (5 мин)

### 1. GitHub Repo
1. Създай нов repo в GitHub (Public)
2. Качи `index.html` и `README.md`

### 2. Cloudflare Pages
1. [dash.cloudflare.com](https://dash.cloudflare.com) → Workers & Pages → Create
2. Pages → Connect to Git → избери repo-то
3. Build command: (празно)
4. Build output directory: (празно или `/`)
5. Save and Deploy

### 3. Embed в Notion
1. Копирай URL-а от Cloudflare (напр. `https://calm-tbd.pages.dev`)
2. В Notion страницата с Road Map-а: `/embed` → постави URL-а

## Без API ключове
Widget-ът пази данните локално в браузъра. Не е нужен Notion API ключ.
